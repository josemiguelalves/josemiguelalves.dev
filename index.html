AWS CLI Configure

A VS Code–compatible extension for Cursor, VSCodium, and other Open VSX–based editors. Open your AWS CLI credentials and config files, switch the active profile from the status bar, and set any named profile as [default] — without using the terminal.

Install on Open VSX

In your editor: Extensions panel → search AWS CLI Configure, or use the link above.
