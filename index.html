AWS CLI Configure

A VS Code–compatible extension to open your AWS CLI credentials and config files, switch the active profile from the status bar, and set any named profile as [default] — no terminal required.

Install

Install on Open VSX (Cursor, VSCodium, & compatible editors)

Open your editor’s Extensions panel, search for AWS CLI Configure, or use the link above for the official listing.
